# AI Агент с памятью (DeepSeek API)

Простой консольный чат-бот на Python, использующий DeepSeek API для генерации ответов.

## Требования

- Python 3.8+
- Библиотека `openai` (OpenAI Python клиент)

## Установка зависимостей

# Install venv if not already installed
```bash
sudo apt install python3-venv
python3 -m venv deepseek-env
source deepseek-env/bin/activate
pip install openai
```


## Запуск скрипта


### Способ 0: Переменная окружения уже добавлена, среда добавлена, openai установлен
```bash
source ../deepseek-env/bin/activate # используем среду из другой папки чтобы не дублировать
python main.py
```



### Способ 1: Установка переменной окружения через экспорт

```bash
export DEEPSEEK_API_KEY="your-api-key-here"
python main.py
```

### Способ 2: Установка в одной строке

```bash
DEEPSEEK_API_KEY="your-api-key-here" python main.py
```

### Способ 3: На Windows (PowerShell)

```powershell
$env:DEEPSEEK_API_KEY="your-api-key-here"
python main.py
```

## Использование

После запуска скрипта вы увидите приглашение `Вы: `. Введите свой вопрос или сообщение:

```

Вы: Привет, как дела?

```

## Структура кода

### Файл `agent.py`

Содержит класс `DeepSeekAgent` — основной класс для управления взаимодействием с DeepSeek API:

- **`__init__(system_prompt)`** — инициализирует агента с системным промптом
- **`send_message(user_text)`** — отправляет сообщение и возвращает ответ

Класс автоматически:
- Получает API ключ из переменной окружения `DEEPSEEK_API_KEY`
- Хранит историю диалога в `self.history`
- Отправляет всю историю в API для сохранения контекста
- Возвращает чистый текст ответа (без структуры сообщений)

### Файл `main.py`

CLI интерфейс, который:
- Импортирует класс `DeepSeekAgent` из `agent.py`
- Создает экземпляр агента
- Запускает бесконечный цикл чата
- Обрабатывает команды `exit` и `quit`
- Ловит исключения и выводит ошибки

## Примечания

- Агент помнит весь контекст разговора и учитывает его при ответах
- Каждый запрос отправляет полную историю диалога на DeepSeek
- Для завершения используйте команды `exit` или `quit`, или нажмите `Ctrl+C`
