# Руководство по тестированию персистентной памяти агента

## Описание изменений

Классу `DeepSeekAgent` добавлена функциональность автоматического сохранения и восстановления истории диалогов:

1. **`_load_history()`** — загружает историю из файла `chat_history.json` при инициализации
2. **`_save_history()`** — сохраняет историю на диск после каждого сообщения
3. **Флаг `history_loaded`** — помогает CLI определить, была ли восстановлена история
4. **Обработка ошибок** — все операции с файлами обернуты в try-except

## Тестовый сценарий

### Сценарий 1: Новый диалог

**Шаг 1:** Убедиться, что файла `chat_history.json` нет
```bash
rm chat_history.json 2>/dev/null || true
python main.py
```

**Ожидаемый результат:**
```
✓ Предыдущая история не найдена. Начат новый диалог.

Агент инициализирован. Введите 'exit' или 'quit' для выхода.

Вы: Привет! Как дела?
Агент: [ответ агента]

Вы: Что ты можешь делать?
Агент: [ответ агента]

Вы: exit
До свидания!
```

**Результат в файле `chat_history.json`:**
```json

  {
    "role": "system",
    "content": "Ты полезный помощник. Отвечай кратко и по существу на русском языке."

    "content": "Привет! Как дела?"
  },
  {
    "role": "assistant",
    "content": "[ответ агента]"

  {
    "role": "user",
    "content": "Что ты можешь делать?"
  },
  {
    "role": "assistant",

  }
```

### Сценарий 2: Восстановление контекста

**Шаг 1:** Запустить агент повторно (файл `chat_history.json` уже существует)
```bash
python main.py
```

**Ожидаемый результат:**
```
✓ Обнаружена история прошлых диалогов. Контекст восстановлен.

Агент инициализирован. Введите 'exit' или 'quit' для выхода.

Вы: А что я говорил перед этим?
Агент: [ответ, который включает контекст предыдущего диалога]

Вы: exit
До свидания!
```

**Проверка:** Агент должен помнить содержание предыдущего диалога и использовать его при ответах.

---

### Сценарий 3: Проверка сохранения при ошибке

**Шаг 1:** Запустить агент и вводить сообщения
```bash
python main.py
```

**Шаг 2:** После нескольких сообщений прервать программу (Ctrl+C)
```
Вы: Первое сообщение
Агент: [ответ]

Вы: Второе сообщение
Агент: [ответ]
^C
```

**Ожидаемый результат:** Все сообщения сохранены в файл. При следующем запуске:
```bash
python main.py
```

История восстановится без потерь.

## Детали реализации

### Параметры инициализации
```python
agent = DeepSeekAgent(
    system_prompt="Кастомный промпт",      # опционально
    history_file="custom_path.json"         # опционально
)
```

### Обработка ошибок

1. **При загрузке истории:**
   - Если файл некорректен (поврежденный JSON): выводится предупреждение, начинается новая история
   - Если файл недоступен: выводится предупреждение, начинается новая история

2. **При сохранении истории:**
   - Если не удалось записать на диск: выводится ошибка, программа продолжает работу
   - История сохраняется в кодировке UTF-8 с красивым форматированием (indent=2)

### Преимущества JSON подхода

✓ Простота — легко читать и редактировать вручную  
✓ Надежность — встроенная поддержка в Python  
✓ Универсальность — можно использовать в других языках/приложениях  
✓ Отладка — видимое содержимое истории  
✓ Масштабируемость — без проблем работает с историей из сотен сообщений

## Команды для быстрого тестирования

```bash
# Удалить старую историю и начать новый диалог
rm chat_history.json

# Запустить агент
python main.py

# Проверить сохраненную историю
cat chat_history.json | python -m json.tool

# Удалить конкретное сообщение из истории (если нужно очистить)
python -c "
import json
with open('chat_history.json', 'r') as f:
    data = json.load(f)
# Оставляем только системный промпт
with open('chat_history.json', 'w') as f:
    json.dump([data[0]], f, ensure_ascii=False, indent=2)
"
```
