# Резюме изменений: Добавление персистентной памяти

## Файл: `agent.py`

### Новые импорты
```python
import json
```

### Изменения в `__init__`
- Добавлены параметры: `history_file` (путь к файлу истории, по умолчанию `"chat_history.json"`)
- Добавлены поля:
  - `self.history_file` — путь к файлу для сохранения
  - `self.history_loaded` — флаг, указывает была ли загружена история
  - `self.default_system_prompt` — сохранение системного промпта для повторной инициализации
- Вызов метода `self._load_history()` при инициализации

### Новый метод `_load_history(self) -> None`
```python
def _load_history(self) -> None:
    """Загружает историю диалога из файла, если он существует."""
```
- Проверяет наличие файла `chat_history.json`
- Если файл существует:
  - Загружает данные в `self.history`
  - Устанавливает флаг `self.history_loaded = True`
- Если файла нет:
  - Инициализирует новую историю с системным промптом
  - Устанавливает флаг `self.history_loaded = False`
- Обработка ошибок: перехватывает `JSONDecodeError` и `IOError`, выводит предупреждение

### Новый метод `_save_history(self) -> None`
```python
def _save_history(self) -> None:
    """Сохраняет текущую историю диалога в файл."""
```
- Записывает `self.history` в JSON файл
- Параметры: `ensure_ascii=False` (поддержка русского), `indent=2` (красивое форматирование)
- Обработка ошибок: перехватывает `IOError`, выводит сообщение об ошибке

### Изменения в `send_message`
- После добавления ответа ассистента в историю добавлена строка:
  ```python
  self._save_history()
  ```
- Это гарантирует, что история сохраняется после каждого обмена сообщениями

## Файл: `main.py`

### Изменения в `main()`
- После инициализации агента добавлена проверка флага `agent.history_loaded`
- Если история была загружена:
  ```
  ✓ Обнаружена история прошлых диалогов. Контекст восстановлен.
  ```
- Если история новая:
  ```
  ✓ Предыдущая история не найдена. Начат новый диалог.
  ```

## Созданные файлы

### `chat_history.json`
Автоматически создается при первом запуске. Содержит всю историю диалогов в формате:
```json
[
  {"role": "system", "content": "..."},
  {"role": "user", "content": "..."},
  {"role": "assistant", "content": "..."},
  ...
]
```

## Ключевые особенности

| Требование | Реализация |
|-----------|-----------|
| Выбор хранилища | JSON (`chat_history.json`) — простой, надежный, универсальный |
| Загрузка истории | `_load_history()` вызывается в `__init__` |
| Сохранение истории | `_save_history()` вызывается в конце `send_message()` |
| Обработка ошибок | try-except блоки для файловых операций |
| Информирование пользователя | Вывод статуса при запуске программы |
| Интеграция | Минимальные изменения, не нарушает существующую логику |

## Примеры использования

### Базовое использование (файл по умолчанию)
```python
agent = DeepSeekAgent()
response = agent.send_message("Привет!")
```

### Кастомный путь к файлу истории
```python
agent = DeepSeekAgent(history_file="/path/to/my/history.json")
response = agent.send_message("Привет!")
```

### Кастомный системный промпт
```python
agent = DeepSeekAgent(
    system_prompt="Ты специалист по Python",
    history_file="python_expert_history.json"
)
response = agent.send_message("Как написать декоратор?")
```
