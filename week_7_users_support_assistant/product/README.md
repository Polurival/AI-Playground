# TaskPilot — управление задачами и проектами

> Вымышленный продукт. Эта документация — синтетический корпус для демо-стенда
> ассистента поддержки (см. SPEC.md в корне пакета). Реального сервиса TaskPilot не существует.

TaskPilot — облачный сервис для управления задачами, проектами и командами. Веб-приложение,
мобильные клиенты (iOS/Android) и публичный REST API.

## Возможности

- **Проекты и задачи** — доски, списки, сроки, ответственные, чек-листы.
- **Команды** — участники, роли (владелец, администратор, участник, наблюдатель).
- **Интеграции** — REST API, вебхуки, импорт из CSV.
- **Безопасность** — вход по паролю, двухфакторная аутентификация (TOTP), корпоративный
  SSO по SAML 2.0 (только на тарифе Business).

## Тарифы

| Тариф | Участников | SSO (SAML) | Вебхуки | Лимит API | Хранилище |
|---|---|---|---|---|---|
| Free | до 5 | нет | нет | 60 запросов/мин | 2 ГБ |
| Pro | до 50 | нет | да | 600 запросов/мин | 100 ГБ |
| Business | без лимита | да | да | 3000 запросов/мин | 1 ТБ |

Подробности — в `docs/billing_plans.md`.

## С чего начать

1. Зарегистрируйтесь на `app.taskpilot.example` и создайте рабочее пространство.
2. Пригласите участников: **Настройки → Команда → Пригласить**.
3. Создайте первый проект и добавьте задачи.
4. Для интеграций выпустите API-ключ: **Настройки → API-ключи** (см. `docs/api_keys.md`).
