# Тарифы и лимиты TaskPilot

## Что входит в каждый тариф

| Возможность | Free | Pro | Business |
|---|---|---|---|
| Участников в рабочем пространстве | до 5 | до 50 | без лимита |
| Проектов | 3 | без лимита | без лимита |
| Хранилище файлов | 2 ГБ | 100 ГБ | 1 ТБ |
| Вход по паролю | да | да | да |
| Двухфакторная аутентификация (TOTP) | да | да | да |
| Корпоративный SSO (SAML 2.0) | **нет** | **нет** | **да** |
| SCIM-провижининг | нет | нет | да |
| Вебхуки | нет | да | да |
| Лимит REST API | 60 запросов/мин | 600 запросов/мин | 3000 запросов/мин |
| Журнал аудита | нет | 30 дней | 1 год |
| Поддержка | почта, 48 часов | почта, 8 часов | приоритетная, 2 часа |

## Корпоративный SSO доступен только на Business

SAML-вход (в том числе через Okta, Azure AD, Keycloak) — возможность тарифа **Business**.
На Free и Pro кнопка «Войти через SSO» показывается, но попытка входа завершается ошибкой
`SSO-402` («SSO недоступен на текущем тарифе»).

Что делать пользователю Free или Pro, который не может войти через SSO:

1. Войдите обычным способом — по адресу электронной почты и паролю. Учётная запись существует
   и работает, ограничен только способ входа.
2. Если вход по SSO нужен всей команде — обновите тариф до Business:
   **Настройки → Тариф → Сменить тариф**. SSO включается сразу после оплаты, повторная
   регистрация не требуется.
3. Настроить SAML может только пользователь с ролью «владелец» или «администратор».

## Двухфакторная аутентификация доступна на всех тарифах

2FA (TOTP) не зависит от тарифа и включается на Free тоже: **Настройки → Безопасность →
Двухфакторная аутентификация**.

## Лимиты API по тарифам

Лимит считается на рабочее пространство, окно — одна минута, счётчик скользящий.
При превышении API возвращает `429 Too Many Requests` (код `API-429`) и заголовок
`Retry-After` в секундах. Подробности — `docs/api_keys.md`.

## Смена тарифа

- Повышение тарифа действует немедленно, оплата пересчитывается пропорционально остатку периода.
- Понижение тарифа применяется со следующего платёжного периода. Если после понижения участников
  больше, чем разрешено новым тарифом, лишние переводятся в статус «наблюдатель» и теряют право
  редактировать задачи.
- Понижение с Business на Pro отключает SSO: пользователи, входившие через SAML, должны задать
  пароль через «Забыли пароль?».

## Оплата

Карта, счёт для юридических лиц (от 10 участников) и годовая оплата со скидкой 20%.
Счета и акты — **Настройки → Тариф → Документы**.
